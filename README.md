SymptiQ: AI-Powered Healthcare Assistant 🩺

SymptiQ is a modern, interactive desktop application designed to act as a preliminary healthcare diagnostic assistant. By leveraging Natural Language Processing (NLP) and Machine Learning, SymptiQ allows users to input their symptoms in plain English and instantly receive AI-driven condition predictions, comprehensive clinical overviews, and localized medical routing.

This project was developed as a comprehensive Design and Innovation initiative to bridge the gap between initial symptom onset and professional medical care.


✨ Core Features

Intelligent Symptom Analysis: Translates natural language user inputs into structured mathematical vectors using TF-IDF.

Predictive AI Engine: Utilizes a trained Multinomial Naive Bayes classification model to instantly predict the top 3 most probable medical conditions based on symptom profiles.

Dynamic Clinical Dashboard: A fully responsive, CustomTkinter-based UI featuring multi-page routing, modern card layouts, and smooth fade-in animations.

Comprehensive Medical Database: Integrates a local, offline dataset to provide immediate access to disease overviews, primary causes, and home care strategies.

Specialist Routing: Automatically identifies the necessary medical specialist for the predicted condition and seamlessly routes the user to a localized Google Search (e.g., "Cardiologist near me").

Patient Intake Forms: Features a structured, scrollable UI for users to manage their foundational medical records, vitals, and clinical history.


🛠️ Technology Stack

Language: Python 3.x

Frontend GUI: customtkinter (Modern UI), PIL (Image processing)

Machine Learning: scikit-learn (MultinomialNB, TfidfVectorizer)

Data Processing: pandas, numpy

Model Serialization: joblib
